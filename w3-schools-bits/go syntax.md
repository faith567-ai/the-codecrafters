## Go syntax
Go syntax consists of the following:
Package declaration and this tells where your code belong e.g the program belong to the main package
Import they are the tools you bring into your code e.g importing files included in the fmt package
Functions this is where you define your tasks, this isthe main func where any code inside the curly bracket will be executed
Statements and expression is a function that is made available from the fmt package is used to output/print text(result)
