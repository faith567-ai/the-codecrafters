## Go Syntax
* Go programs have some main parts:
- Package: This shows where the program belongs.

- Example: main means it is the main program.

- Import: This brings in tools you need.

- Example: fmt helps you print text.

- Function: This is where you write what the program will do.

- The main function is where the program starts.

- Statements and expressions: These are the lines of code inside a function, they tell the computer what to do, like printing text on the screen.
