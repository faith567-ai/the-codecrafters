## in GO there are diffrent types of variables e.g

int ; there are whole numbers without decimals 
float32 ; there are numbers with decimals
bool ; options with only true/false
string ; there are words between double quotation marks.
    There are two ways in declaring variables in GO, you can declare using the var keyword or you can use the short variable declaration :=
    so now when they say create a variable and give it a value you go like this:
           book := faith
           book now is the variable name you choose
           := tells GO to create the variables
           faith now is the value(the stored data inside)
           var can be used inside and outside functions
           := can be use only inside functions.
